INSTRUCCIONES DE USO
1. Selecciona la cantidad de colores deseados dando click en el recuadro azul.
2. selecciona el formato del color, ya sea HEX o RGB
3. Haz click en "Generar paleta"
Dependiendo la cantidad de colores seleccionados veras unos cuadros aparecer. 
4. Si un color generado es deseado, clickea el color. 
5. Veras una palabra dentro del cuadro llamada "Bloqueado", significa que ese color no va a cambiar cuando presiones "Generar paleta" de nuevo. 
6. Puedes remover el bloqueo presionando el color bloqueado de nuevo.
7. Puedes bloquear los colores que quieras hasta encontrar los colores ideales.

DECISIONES TECNICAS
La pagina fue creada con el objetivo de ser intuitiva y sencilla de usar. 
Un solo boton interactivo (crearColores) conectado al documento .js.
Una etiqueta (label) con las opciones "6", "8" y "9" para elegir la cantidad de colores solicitados.
Otra etiqueta label esta vez con dos opciones: "HEX" y "RGB" para elegir en que formato se requiere en la generacion de colores.
Finalmente con atributos aria se creo la funcion de feedback para confirmar el cambio hecho en la pagina web

PASOS PARA EJECUTAR Y DESPLEGAR APLICACION.
La aplicacion se ejecuta solo presionando el boton principal "generar paleta" y se ddesplega sencillamente cerrando el navegador.
